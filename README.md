## 📊 Project Overview
1. Analyses various media over Netflix
2. Compares movies and series ratio
3. Understands the most famous genre and geographical distribution

## 💡 Key Insights & Findings
1. Top Genre - Drama
2. Top Country - USA
3. Number of movies released is more than the number of series
